# NFT Upload Landing Page

## Overview
This project is a landing page that allows users to upload their NFTs (Non-Fungible Tokens). It provides a user-friendly interface for users to submit their digital assets easily.

## Project Structure
```
nft-upload-landing
├── src
│   ├── index.html          # Main HTML document for the landing page
│   ├── css
│   │   └── styles.css      # Styles for the landing page
│   ├── js
│   │   ├── main.js         # Main JavaScript entry point
│   │   └── nft-upload.js   # JavaScript logic for NFT upload feature
│   └── assets
│       └── images          # Directory for image assets
├── package.json            # npm configuration file
└── README.md               # Project documentation
```

## Features
- User-friendly interface for uploading NFTs.
- File input validation to ensure correct file types are uploaded.
- Responsive design for optimal viewing on various devices.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd nft-upload-landing
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Open `src/index.html` in your web browser to view the landing page.

## Usage
- Users can select files to upload their NFTs.
- The application will validate the file type and size before submission.
- Upon successful upload, users will receive a confirmation message.
